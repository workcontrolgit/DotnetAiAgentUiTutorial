Choice [1]: █
Auto-selecting 1 in 2s…

How it works

  • After connecting to MCP, Program.cs shows the style picker using AnsiConsole.Prompt
  • A 2-second countdown using AnsiConsole.Status auto-selects Style 1 (Structured) if no key is pressed
  • The chosen UiStyle enum value is passed into new HrAgent(chatClient, mcpTools, style)
  • The enum lives in HrAgent.cs: public enum UiStyle { Structured, Minimal, Panels }
