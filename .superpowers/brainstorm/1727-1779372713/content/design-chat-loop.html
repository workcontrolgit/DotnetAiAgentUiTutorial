HrAgent.cs changes

  • Constructor gains a UiStyle style = UiStyle.Structured parameter
  • RunAsync calls private helpers: RenderUserPrompt(), RenderToolCall(name), RenderResponse(text)
  • Each helper switches on _style — one switch expression per helper (~10 lines each)
  • Tool call spinner uses AnsiConsole.Status().Start() wrapping chatClient.GetResponseAsync
  • Style A response: AnsiConsole.Write(new Table(...)) if response contains structured position data; falls back to plain markup otherwise
  • Style B response: AnsiConsole.Write(new Panel(text).BorderColor(Color.Teal))
  • Style C response: AnsiConsole.Write(new Panel(text).Header("ASSISTANT").BorderColor(Color.MediumAquamarine))
