Complete picture of what changes and what stays the same

Files changed

  • HrMcp.Agent.csproj — add Spectre.Console 0.49.*
  • HrAgent.cs — add UiStyle enum, style constructor param, 3 private render helpers, tool call spinner, switch on style in RunAsync
  • Program.cs — add startup style picker (prompt + 2s auto-select countdown) before constructing HrAgent
